# Shared Network Outputs

`stacks/dev/us-east-1/shared-network-foundation` exports:

- `dev-us-east-1-LogsKeyArn`
- `dev-us-east-1-NetworkLogsBucketName`
- `dev-us-east-1-SharedLogGroupName`
- `dev-us-east-1-FlowLogsRoleArn`

These values are consumed by `observability` and any future network or inspection logging stacks.
